# Team Stock

チームで在庫を管理するためのシンプルなウェブアプリケーションです。複数のチームメンバーが共同で在庫アイテムを管理できます。

## 概要

Team Stockは、チームベースの在庫管理を簡単に行うためのツールです。以下の機能を提供します。

- Google認証によるシームレスなログイン
- 複数チームの作成と管理
- チームメンバーの招待
- 在庫アイテムの追加・編集・削除
- タグによる在庫の分類とフィルタリング
- 直感的なダッシュボード

## セットアップ

### 前提条件

- Node.js 22.13.1以上
- npm
- Google OAuth認証情報(Google Cloud Platformで取得)
